How To Do Research

The PIC32 can be a daunting beast. When you don’t know what the next step is, some of the following should help.

Online Microchip Documents
www.microchip.com and search for your chip

  • The Datasheet
    • This should always be the first place to look. Get comfortable with it.
  • Reference Manuals
    • These are more in depth and specific than the datasheet
  • Code examples
    • Starting from scratch can be exhausting…
  • Application Notes
    • These are someone else’s documented projects

UMassAmherstM5.org
The shameless plug

  • Tech Tutorials -> PIC32 Tutorials

MPLAB X Online Help
Come here for help with ‘config bits’ and ‘peripheral libraries’ in particular

  • Help -> Help Contents
  • The sections under ‘C32 Toolchain’ have proven particularly useful

The Peripheral Library
These are a set of functions written by Microchip for simpler code (in theory)

  • Google ‘PIC32 Peripheral Libraries for MPLAB C32 Compiler’
    • …or just use this link
  • Note that I can’t fully recommend these as they seem to cause more issues than they solve
  • Also note that you may encounter functions not in the pdf linked to above that will work (sometimes only with a certain compiler>

Books
Why of course!

Programming 32-bit Microcontrollers in C: Exploring the PIC32 by Lucio Di Jasio
This would be my recommended book for working with any PIC32. Note that it was written for the first family of PIC32’s (3xx/4xx) as well as for MPLAB and not MPLAB X so some information doesn’t translate well. What is in the book is rewarding and a pleasure to read for PIC32’s and really any embedded system. There is also an associated website (Exploring the PIC32) that deals with some of the changes made for PIC32’s after the books release.

Designing Embedded Hardware by John Catsoulis
This is one of the best O’Reilly books I’ve read. For a while, I read it before falling asleep at night. It’s a pleasure and provides a wealth of approachable embedded hardware information. Not focused on PIC microcontrollers but useful for any embedded development.
